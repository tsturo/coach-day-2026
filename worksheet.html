Task 01 — Peel back the layers.
Coach Day · AI — Expectations vs. Reality · 15 min
Group
Case
How to use this sheet
  • Start with the Surface — what's visible from the outside, what a manager notices in a status report.
  • Then ask "why?" at least three times. Each answer becomes the next question. Write what comes out in Underneath.
  • Don't stop until you reach the Core — a fear or need about a real human: identity, belonging, control, safety, meaning, status.
  • If the Core column still reads like a process problem, you haven't gone deep enough. Keep going.
Surface
What's visible? What would a manager see from the outside — in standups, metrics, Slack, the PR queue?
Underneath
What's driving it? Which systems, incentives, norms, or histories produced this surface? What would stop if those went away?
Core
What is this person actually afraid of losing, or needing to protect? Name it in human terms — one sentence.
Forcing function
If the Core column still reads like a process problem ("reviews are slow", "no feedback loop"), you haven't gone deep enough. The core is always about a person: identity, safety, meaning, belonging, status, control.

Task 02 — Stop. Start. Escalate.
Coach Day · AI — Expectations vs. Reality · 30 min
Group
Case
How to use this sheet
  • 5 min — draft Stop items and candidate Escalate items together (quick bullets).
  • 10 min — brainstorm Start action points: practices, rituals, guardrails, experiments, pilots. Get several on paper.
  • 5 min silent — each person writes their own opening lines to the person in the case. Full sentences, your own words.
  • 8 min — read versions aloud, merge the strongest one into the Start column. Then write what you'd NOT say below it.
  • 2 min — lock one Escalate item with a specific name and date.
Stop
What are we currently doing that makes this problem worse? Think: habits, rituals, defaults, unspoken rules, review practices, language we use.
Start
New action points: a practice, ritual, guardrail, small experiment, working agreement, visible change in what gets praised. One action must be the actual opening of the conversation you'd have on Monday — full sentences, your own words.
Escalate
What can't we fix at the coach level? Name the specific person with the authority to decide, what we're asking them, and when we need it.
The ask (what we need them to do)
Who (name + role)
By when (date)
Forcing functions
The conversation must only make sense for THIS case. If it would work for any other case, it's too generic — go back and make it specific. "Escalate" needs a name and a date. Not "leadership should communicate better" — "we bring this to [name] by [date]."
